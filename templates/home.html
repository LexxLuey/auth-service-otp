🔐 Authentication Service with OTP

Welcome to the Email-based OTP Authentication Service

This API provides secure authentication using one-time passwords with:

⚡ Redis Rate Limiting 🔄 Celery Async Tasks 🔑 JWT Tokens 📊 Audit Logging
📚 API Root: /api/v1/
📓 API Docs: /api/docs/
❤️ Health Check: /api/v1/health
🔑 Request OTP: POST /api/v1/auth/otp/request
🔓 Verify OTP: POST /api/v1/auth/otp/verify
📝 View Audit Logs: GET /api/v1/audit/logs
Built with Django & DRF • OTP Authentication Service • 🐍 v1.0
